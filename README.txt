**HELLO MY DAY 어플리케이션**


- 기능 구성
1. D-DAY 날짜 계산
2. 메모장 작성, 수정, 삭제 기능
3. 현재 위치 표시

-코드 구성
1. main : 기능 동작 코드 
2. res : layout폴더 내 xml 코드 , 코드 실행에 필요한 기본 리소스들 
3. build.gradle : 빌드 과정
4. AndroidManifest: 전체 레이아웃 및 동작 실행

-어플리케이션 사용 방법
1. D-Day 계산
  D-Day 버튼 클릭 -> 날짜 선택 -> D-Day 출력

2. 날짜, 시간 및 캘린더
  오늘 날짜 및 현재시간 확인 가능, 하단 달력으로 캘린더 확인 가능

3. 메모 기능
  -메모 입력 및 저장 : 'MEMO'버튼 클릭 -> 내용 입력 -> '저장'버튼 클릭 
  -저장한 메모 불러오기 : '불러오기'버튼 클릭
  -저장한 메모 삭제하기 : '불러오기'버튼 클릭 -> '삭제'버튼 클릭
